# 回归测试命题

用于检查 `xuanze-between` skill 是否出现主题误判或模板串用。

## 固定测试集

1. 如果 AI 比人类更懂你，你还需要真实的人际关系吗？
   - 期望识别：AI 技术 / 稳定回应 vs 真实他者

2. 如果死亡让生命变得有意义，那永生是不是一种惩罚？
   - 期望识别：死亡意义 / 有限性是否赋予意义

3. 如果救一个陌生人会伤害你最亲近的人，你还会救吗？
   - 期望识别：道德救助 / 普遍伦理 vs 特殊责任

4. 如果爱情会让你失去自由，你还会选择它吗？
   - 期望识别：亲密关系 / 爱与自由

5. 如果安全必须以自由为代价，你愿意被保护吗？
   - 期望识别：自由安全 / 被保护 vs 被管理

6. 如果努力只是让你更适应规则，而不是更自由，你还要努力吗？
   - 期望识别：成长未来 / 努力 vs 驯化

7. 如果一个人能删除痛苦记忆，他还算完整的自己吗？
   - 期望识别：记忆痛苦 / 完整自我 vs 减少痛苦
