# 好样例：死亡意义 / 永生

输入：
如果死亡让生命变得有意义，那永生是不是一种惩罚？

判断：
核心不是“害怕死亡”，而是“有限性是否赋予意义”。A/B 必须正面回答永生是不是惩罚。

输出要点：
- 标题：永生，是奖励还是惩罚？
- 识别主题：死亡意义 / 终点赋予价值 vs 无限延展带来可能
- A：永生是一种惩罚。因为没有终点，选择会失去重量，时间会把一切珍贵慢慢磨平。
- B：永生不一定是惩罚。因为意义不只来自死亡，也可以来自持续创造、重新选择和不断理解世界。
- 追问围绕：无限时间是否稀释今天、死亡消失后爱与遗憾是否仍珍贵、永生是自由还是囚禁。
- 分镜母题：白色病房、静止钟表、无限走廊、生日蜡烛、停止的沙漏、清晨墓园。
