# 坏样例：不要把自我保护错写成爱与自由

输入：
如果一个人越来越懂得保护自己，也越来越难爱别人，这算成长吗？

错误判断：
把它写成“爱情会夺走自由吗？”或“爱与自由 / 亲密承诺 vs 个体边界”。

为什么错：
- 用户问的是自我保护的代价，不是爱情是否限制自由。
- 核心冲突应是“自我保护 vs 情感开放”或“免于受伤 vs 仍能相爱”。
- A/B 应正面回答“这算不算成长”，而不是回答“还要不要选择爱情”。

错误输出特征：
- A：仍然选择爱情，因为真正的爱不是夺走自由。
- B：不选择这种爱情，因为它是温柔占有。
- 分镜出现雨中车站、你在哪、被占有等母题。

更合适的方向：
- 标题：保护自己之后
- 识别主题：亲密关系 / 自我保护
- A：这算成长。因为懂得保护自己，意味着不再把被爱当作唯一出口。
- B：这不完全算成长。因为如果保护自己变成无法靠近任何人，它也可能是一种新的封闭。
- 分镜母题：锁上的门、未发出的消息、撤回的手、玻璃后的拥抱、安静房间里的防备。
